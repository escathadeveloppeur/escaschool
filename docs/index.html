Comment ça marche
Création d'école & code d'activation
Chaque établissement reçoit un code unique généré par nos soins. Si votre école n'est pas encore enregistrée, nous vous fournissons ce code pour activer votre instance.

1. Demande de code

L'école contacte ESC SERVICES pour obtenir son code d'activation. Nous vérifions l'existence de l'établissement et générons un code unique.

2. Activation

L'administrateur de l'école entre le code dans l'application. L'école est alors activée et prête à être configurée (classes, élèves, enseignants, etc.).

FAQ
Questions fréquentes
Tout ce que vous devez savoir avant de démarrer.
Comment obtenir le code d'activation pour mon école ?
Contactez-nous par téléphone au +243 813 012 087 ou par email. Nous vérifions les informations de votre établissement et vous envoyons un code unique sous 24h.
Le paiement est-il vraiment de 0,66666666 $ par élève par mois ?
Oui, le tarif est fixe et universel. Vous ne payez que pour le nombre d'élèves actifs. La facturation est annuelle, ce qui revient à 8 $ par élève par an.
L'application fonctionne-t-elle hors ligne ?
EscaSchool est conçue pour fonctionner en mode connecté, mais des fonctionnalités de base (consultation des données) sont disponibles hors ligne. La synchronisation se fait automatiquement dès que la connexion est rétablie.
Quelles sont les langues disponibles ?
Actuellement, l'application est disponible en français, anglais, espagnol et portugais. D'autres langues seront ajoutées prochainement.
